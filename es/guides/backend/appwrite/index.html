Appwrite y Astro

Appwrite es una plataforma de backend como servicio (BaaS) autohospedada que ofrece autenticación y gestión de cuentas, preferencias de usuario, persistencia de base de datos y almacenamiento, funciones en la nube, localización, manipulación de imágenes y otras utilidades del lado del servidor.
