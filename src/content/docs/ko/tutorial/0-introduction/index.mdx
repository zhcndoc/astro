<Lede> 이 튜토리얼에서는 처음부터 정식 출시까지 모든 기능을 갖춘 블로그를 구축하여 Astro의 주요 기능을 배우게 됩니다! 🚀 </Lede>

그 과정에서 다음을 수행하게 됩니다. 
- 개발 환경 설정
- 웹사이트에 페이지 및 블로그 게시물 만들기
- Astro 컴포넌트로 빌드
- 로컬 파일 쿼리 및 작업
- 사이트에 상호작용 추가
- 사이트를 웹에 배포

무엇을 만들 것인지 미리 보고 싶으십니까? [GitHub](https://github.com/withastro/blog-tutorial-demo) 또는 [StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/complete?file=src%2Fpages%2Findex.astro)에서 최종 프로젝트를 볼 수 있습니다. 

:::note
미리 구축된 Astro 사이트를 찾고 있다면 https://astro.new 를 방문하여 시작 템플릿을 선택해 온라인 편집기에서 열고 편집하세요.
:::
