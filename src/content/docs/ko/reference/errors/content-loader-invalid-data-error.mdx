> **오류 메시지 예시:**<br/>
The loader for **blog** returned invalid data.<br/>
Object is missing required property "id".

## 무엇이 잘못되었나요?
콘텐츠 컬렉션의 로더가 잘못된 데이터를 반환했습니다.
인라인 로더는 고유 ID 필드가 있는 객체의 배열 또는 ID를 키로, 항목을 값으로 사용하는 일반 객체를 반환해야 합니다.
