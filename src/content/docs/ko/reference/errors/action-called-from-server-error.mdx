> **ActionCalledFromServerError**: Action called from a server page or endpoint without using `Astro.callAction()`.

## 무엇이 잘못되었나요?
`Astro.callAction()`을 사용하지 않고 서버 페이지 또는 엔드포인트에서 호출된 액션입니다.
