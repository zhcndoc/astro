> **ReservedSlotName**: Unable to create a slot named `SLOT_NAME`. `SLOT_NAME` is a reserved slot name. Please update the name of this slot.

## 무엇이 잘못되었나요?

일부 단어는 내부적으로 이미 사용되고 있으므로 슬롯 이름에 사용할 수 없습니다.
