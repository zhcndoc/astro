> **ImageMissingAlt**: Image missing "alt" property. "alt" text is required to describe important images on the page.

## 무엇이 잘못되었나요?

`alt` 속성을 사용하면 스크린 리더 및 기타 보조 기술 사용자에게 설명적인 대체 텍스트를 제공할 수 있습니다. 이미지에 액세스할 수 있도록 하려면 `Image` 컴포넌트에 `alt`를 지정해야 합니다.

이미지가 단지 장식용인 경우 (즉, 페이지 이해에 도움이 되지 않는 경우) 스크린 리더가 이미지를 무시할 수 있도록 `alt=""`를 설정하세요.
