> **GetStaticPathsExpectedParams**: Missing or empty required `params` property on `getStaticPaths` route.

## 무엇이 잘못되었나요?

`getStaticPaths`로 지정된 모든 경로에는 경로와 일치하는 데 필요한 경로 매개변수를 지정하는 `params` 속성이 필요합니다.

예를 들어 코드는 다음과 같습니다.

```astro title="pages/blog/[id].astro"
---
export async function getStaticPaths() {
	return [
		{ params: { id: '1' } }
	];
}
---
```

다음 경로를 생성합니다: `site.com/blog/1`.
