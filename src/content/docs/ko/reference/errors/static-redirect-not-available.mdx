:::caution[사용되지 않음]
버전 2.6부터 더 이상 사용되지 않습니다.
:::

> **StaticRedirectNotAvailable**: Redirects are only available when using `output: 'server'` or `output: 'hybrid'`. Update your Astro config if you need SSR features.

## 무엇이 잘못되었나요?

`Astro.redirect` 함수는 [서버 측 렌더링](/ko/guides/on-demand-rendering/)이 활성화된 경우에만 사용할 수 있습니다.

정적 웹사이트에서 리디렉션하려면 [meta refresh 속성](https://developer.mozilla.org/ko/docs/Web/HTML/Element/meta)을 사용할 수 있습니다. 특정 호스트는 구성 기반 리디렉션 (예: [Netlify redirects](https://docs.netlify.com/routing/redirects/))도 제공합니다.
