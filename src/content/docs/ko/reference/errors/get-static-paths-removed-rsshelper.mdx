:::caution[사용되지 않음]
Astro 4.0부터 더 이상 사용되지 않습니다. 오류 대체가 있는 RSS 도우미는 더 이상 존재하지 않습니다.
:::

> **GetStaticPathsRemovedRSSHelper**: The RSS helper has been removed from `getStaticPaths`. Try the new @astrojs/rss package instead.

## 무엇이 잘못되었나요?

`getStaticPaths`는 더 이상 RSS 피드 생성을 위한 도우미를 노출하지 않습니다. 대신 [@astrojs/rss](/ko/guides/rss/#astrojsrss-설정) 통합으로 마이그레이션하는 것이 좋습니다.
