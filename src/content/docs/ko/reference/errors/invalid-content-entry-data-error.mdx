> **오류 메시지 예시:**<br/>
**blog** → **post** frontmatter does not match collection schema.<br/>
"title" is required.<br/>
"date" must be a valid date.

## 무엇이 잘못되었나요?
콘텐츠 항목이 컬렉션 스키마와 일치하지 않습니다.
모든 필수 필드가 있는지, 모든 필드의 타입이 올바른지 확인하세요.
`src/content.config.*` 파일에서 컬렉션 스키마와 대조하여 확인할 수 있습니다.
자세한 내용은 [콘텐츠 컬렉션 문서](/ko/guides/content-collections/)를 참조하세요.
