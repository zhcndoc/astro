호환되는 편집기에서 콘텐츠 컬렉션 항목에 대한 자동 완성 기능(예: 코드 완성, 빠른 힌트)을 활성화합니다.

활성화하면 이 기능은 프로젝트의 `.astro` 디렉터리에 JSON 스키마를 생성하고 추가합니다. 이러한 파일들은 Astro 언어 서버가 콘텐츠 파일(`.md`, `.mdx`, `.mdoc`)에서 자동 완성을 제공하는 데 사용될 수 있습니다.

```js
{
  experimental: {
    contentIntellisense: true,
  },
}
```

Astro VS Code 확장에서 이 기능을 사용하려면 VS Code 설정에서 `astro.content-intellisense` 옵션도 활성화해야 합니다. Astro 언어 서버를 직접 사용하는 편집기의 경우, 이 기능을 활성화하기 위해 초기화 매개변수로 `contentIntellisense: true`를 전달하세요.
