프런트엔드 개발자를 위해 설계된 단일 명령 웹 게시 플랫폼인 [Surge](https://surge.sh/)에 Astro 프로젝트를 배포할 수 있습니다.

## 배포 방법

<Steps>
1. [Surge CLI](https://www.npmjs.com/package/surge)를 아직 설치하지 않았다면 터미널에서 전역으로 설치하세요.

    ```shell
    npm install -g surge
    ```

2. 프로젝트의 루트 디렉터리에서 Astro 사이트를 빌드하세요.

    ```shell
    npm run build
    ```

3. CLI를 사용하여 Surge에 배포하세요.

    ```shell
    surge dist
    ```

    `surge dist yourdomain.com`을 실행하여 배포할 때 [Surge에서 사용자 정의 도메인](http://surge.sh/help/adding-a-custom-domain)을 사용할 수 있습니다.
</Steps>
