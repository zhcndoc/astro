[Fleek](http://fleek.xyz/)을 사용하여 정적 Astro 사이트를 에지에 최적화된 분산형 네트워크에 배포할 수 있습니다.

이 가이드는 Fleek UI와 CLI를 사용하여 Astro 사이트를 Fleek에 배포하는 방법을 자세히 설명합니다.

## 프로젝트 구성

Astro 프로젝트는 정적 사이트로 Fleek에 배포할 수 있습니다.

## 배포 방법

웹사이트 UI를 통해 또는 Fleek의 CLI (명령줄 인터페이스)를 사용하여 Fleek에 배포할 수 있습니다.

### 플랫폼 UI 배포

<Steps>
1. [Fleek](https://app.fleek.xyz) 계정을 만듭니다.

2. 코드를 온라인 Git 저장소 (GitHub)에 푸시합니다.

3. 프로젝트를 Fleek으로 가져옵니다.

4. Fleek이 자동으로 Astro를 감지한 다음 올바른 설정을 구성할 수 있습니다.

5. 애플리케이션이 배포되었습니다!
</Steps>


### Fleek CLI

<Steps>
1. Fleek CLI를 설치합니다.

    ```bash
    # Nodejs >= 18.18.2가 필요합니다.
    npm install -g @fleek-platform/cli
    ```
2. 터미널에서 Fleek 계정에 로그인합니다.

    ```bash
    fleek login
    ```

3. 빌드 명령을 실행하여 정적 파일을 생성합니다. 기본적으로 이러한 파일은 `dist/` 디렉터리에 위치합니다.

    ```bash
    npm run build
    ```

4. 프로젝트를 초기화합니다. 그러면 구성 파일이 생성됩니다.

    ```bash
    fleek sites init
    ```

5. 새 Fleek 사이트를 만들거나 기존 사이트를 사용하라는 메시지가 표시됩니다. 사이트 이름을 지정하고 프로젝트가 있는 디렉터리를 선택합니다.

6. 사이트를 배포합니다.

    ```bash
    fleek sites deploy
    ```

</Steps>
