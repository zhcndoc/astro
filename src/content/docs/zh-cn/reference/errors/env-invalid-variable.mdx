> **EnvInvalidVariable**: 以下环境变量与 `experimental.env.schema` 中定义的数据类型和/或属性不匹配：KEY 不是 TYPE 类型

## 发生了什么错误？
一个环境变量与 `experimental.env.schema` 中定义的数据类型和/或属性不匹配。
