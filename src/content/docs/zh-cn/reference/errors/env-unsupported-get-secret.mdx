:::caution[已废弃]
这个错误来自 Astro 的旧版本，现在已不再使用。如果你无法将项目升级到更近期的版本，那么你可以查阅 [不再维护的旧文档快照](/zh-cn/upgrade-astro/#旧版本文档不再维护) 以获取帮助。
:::

> **EnvUnsupportedGetSecret**: `astro:env/server` 模块导出的函数 `getSecret` 不被你的适配器支持。

## 发生了什么错误？
`astro:env/server` 模块导出的函数 `getSecret()` 不被你的适配器支持。
