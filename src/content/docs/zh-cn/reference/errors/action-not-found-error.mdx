> **ActionNotFoundError**: 服务器接收到一个名为 `ACTION_NAME` 的 action 请求，但找不到匹配项。如果你重命名了 action，请检查你是否已经更新为匹配的 `actions/index` 文件和调用代码。

## 哪里出了问题？
服务器接收到了 action 的请求，但是无法找到同名的匹配项。
