> **ActionsReturnedInvalidDataError**: Action handler 返回了无效数据。handler 应该返回可序列化的数据类型，例如对象、数组、字符串和数字。解析错误：**错误**

## 哪里出了问题？
Action handler 返回了无效数据。handler 应该返回可序列化的数据类型，并且不能返回 Response 对象。
