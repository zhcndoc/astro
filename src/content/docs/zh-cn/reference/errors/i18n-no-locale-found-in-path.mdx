> **i18nNoLocaleFoundInPath**: 你尝试在不包含语言环境的路径上使用 i18n 工具。你可以先使用 `pathHasLocale` 来判断路径中是否包含语言环境。

## 出了什么问题？
一个 i18n 工具试图从一个不包含语言环境的 URL 路径中获取语言环境。你可以通过先使用 pathHasLocale 来检查 URL 是否包含语言环境，以防止此错误。
