> **i18nNoLocaleFoundInPath**: Você tentou usar uma utilidade i18n em um caminho que não contém qualquer local. Você pode usar `pathHasLocale` antes para determinar se o caminho possui algum local.

## O que deu errado?
Uma utilidade i18n tentou usar o local de um caminho de URL que não possui nenhum. Você pode prevenir esse erro usando pathHasLocale para verificar URLs de um local antes de usar utilidades i18n.
