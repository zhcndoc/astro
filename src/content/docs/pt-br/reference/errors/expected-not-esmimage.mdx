> **ExpectedNotESMImage**: Uma imagem importada via ESM não pode ser passada diretamente para `getImage()`. Em vez disso, passe um objeto com a imagem na propriedade `src`.

## O que deu errado?
Uma imagem importada via ESM não pode ser passada diretamente para `getImage()`. Em vez disso, passe um objeto com a imagem na propriedade `src`.

```diff
import { getImage } from "astro:assets";
import myImage from "../assets/my_image.png";
- const optimizedImage = await getImage( myImage );
+ const optimizedImage = await getImage({ src: myImage });
```
