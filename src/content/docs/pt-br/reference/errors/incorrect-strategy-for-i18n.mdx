> **IncorrectStrategyForI18n**: A função `FUNCTION_NAME` pode ser usada somente quando `i18n.routing.strategy` é configurado como `"manual"`.

## O que deu errado?
Algumas funções de internacionalização são disponíveis apenas quando o próprio roteamento i18n Astro está desabilitado pelo ajuste de configuração `i18n.routing: "manual"`.
