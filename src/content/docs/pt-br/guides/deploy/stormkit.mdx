Você pode publicar seu projeto Astro na [Stormkit](https://stormkit.io/), uma plataforma de implantação para websites estáticos, aplicações de página única (SPAs), e funções _serverless_.

## Como publicar

<Steps>
1. [Entre na Stormkit](https://app.stormkit.io/auth).

2. Pela interface de usuário, importe seu projeto Astro de um dos três provedores Git suportados (GitHub, GitLab, ou Bitbucket).

3. Navegue ao ambiente de produção do projeto na Stormkit ou crie um novo ambiente se necessário.

4. Verifique o comando de build na sua [configuração Stormkit](https://stormkit.io/docs/deployments/configuration). Por padrão, a CI Stormkit executará `npm run build` mas você pode especificar um comando de build personalizado nessa página.

5. Clique no botão "Publicar Agora" para pulicar seu site.
</Steps>

<ReadMore>Leia mais na [Documentaçõa Stormkit](https://stormkit.io/docs).</ReadMore>
