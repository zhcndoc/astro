[Zeabur](https://zeabur.com) oferece hospedagem para aplicações web full-stack. Sites Astro podem ser hospedados tanto com SSR quanto como estáticos.

Este guia inclui instruções para publicar na Zeabur através da interface do site.

## Configuração do Projeto

### Site Estático

Astro gera um site estático por padrão. Não é necessário nenhuma configuração extra para publicar um site Astro estático na Zeabur.

### Adaptador para SSR

Para habilitar SSR no seu projeto Astro e publicar na Zeabur:

<Steps>
1. Instale [o adaptador `@zeabur/astro-adapter`](https://www.npmjs.com/package/@zeabur/astro-adapter) nas dependências do seu projeto usando o gerenciador de pacotes de sua preferência. Se você está usando npm ou não tem certeza, execute este comando no terminal:

    ```bash
      npm install @zeabur/astro-adapter
    ```

2. Adicione duas novas linhas no arquivo de configuração `astro.config.mjs` do seu projeto .

    ```js title="astro.config.mjs" ins={2, 5-6}
    import { defineConfig } from 'astro/config';
    import zeabur from '@zeabur/astro-adapter/serverless';

    export default defineConfig({
      output: 'server',
      adapter: zeabur(),
    });
    ```
</Steps>

## Como publicar

Você pode publicar seu site Astro na Zeabur se o projeto estiver armazenado no GitHub.

<Steps>
1. Clique em <kbd>Create new project</kbd> no [painel Zeabur](https://dash.zeabur.com).

2. Configure a instalação do GitHub e importe o repositório.

3. Zeabur detectará automaticamente que seu projeto é um projeto Astro e o construirá usando o comando `astro build`.

4. Assim que a construção for concluída, você pode vincular um domínio ao seu site e visitá-lo.
</Steps>

Após o projeto ser importado e publicado, todos os futuros _pushes_ para branches gerarão novas builds.

Saiba mais sobre no [Guia de Publicação](https://zeabur.com/docs/get-started/) Zeabur.
