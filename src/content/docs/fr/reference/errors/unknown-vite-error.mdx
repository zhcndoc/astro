## Qu'est-ce qui a mal tourné ?
Vite a rencontré une erreur inconnue lors du rendu de votre projet. Nous ne savons malheureusement pas ce qui s'est passé (sinon nous vous le dirions !)

Si vous pouvez provoquer cette erreur de manière fiable, nous vous serions reconnaissants de bien vouloir [ouvrir un ticket](https://astro.build/issues/).

-  [Guide de dépannage de Vite](https://vite.dev/guide/troubleshooting.html)
