> **ContentSchemaContainsSlugError**: A content collection schema should not contain `slug` since it is reserved for slug generation. Remove this from your COLLECTION_NAME collection schema.

## Qu'est-ce qui a mal tourné ?
Un schéma de collection de contenu hérité ne doit pas contenir le champ `slug`. Ce champ est réservé par Astro pour la génération d'entrées. Supprimez le champ `slug` de votre schéma, ou choisissez un nom différent.
