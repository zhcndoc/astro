> **AdapterSupportOutputMismatch**: The `ADAPTER_NAME` adapter is configured to output a static website, but the project contains server-rendered pages. Please install and configure the appropriate server adapter for your final deployment.

## Qu'est-ce qui a mal tourné ?
L'adaptateur actuellement configuré ne prend pas en charge le rendu côté serveur, qui est requis pour la configuration actuelle du projet.

En fonction de votre adaptateur, il peut y avoir un point d'entrée différent à utiliser pour le rendu côté serveur. Par exemple, l'adaptateur `@astrojs/vercel` possède un point d'entrée `@astrojs/vercel/static` pour le rendu statique et un point d'entrée `@astrojs/vercel/serverless` pour le rendu côté serveur.
