> **NoAdapterInstalled**: Cannot use `output: 'server'` or `output: 'hybrid'` without an adapter. Please install and configure the appropriate server adapter for your final deployment.

## Qu'est-ce qui a mal tourné ?
Pour utiliser le rendu côté serveur, un adaptateur doit être installé afin qu'Astro sache comment générer la sortie appropriée pour la plateforme de déploiement ciblée.
