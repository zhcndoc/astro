Теперь, когда вы можете строить с помощью компонентов, пришло время создать несколько пользовательских макетов!

## Забегая вперед

В этом модуле вы создадите макеты для общих элементов и стилей на страницах и в записях блога.


Для этого вы будете:

- Создавать многоразовые компоненты макета
- Передавать контент в ваши макеты с помощью `<slot />`
- Передавать данные из frontmatter Markdown в макеты
- Вкладывать несколько макетов
