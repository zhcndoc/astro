> **MissingSharp**: Не удалось найти Sharp. Пожалуйста, установите Sharp (`sharp`) вручную в ваш проект или перейдите на другую службу изображений.

## Что пошло не так?

Sharp - это служба обработки изображений по умолчанию, используемая для `astro:assets`. При использовании [строгого менеджера пакетов](https://pnpm.io/pnpm-vs-npm#npms-flat-tree), такого как pnpm, Sharp должен быть установлен вручную в ваш проект, чтобы использовать обработку изображений.

Если вы не используете `astro:assets` для обработки изображений и не хотите устанавливать Sharp, вы можете настроить следующую службу обработки изображений, которая не выполняет обработку:

```js
import { defineConfig, passthroughImageService } from "astro/config";
export default defineConfig({
 image: {
   service: passthroughImageService(),
 },
});
```
