> **AstroGlobUsedOutside**: `Astro.glob(GLOB_STR)` можно использовать только в файлах `.astro`. Вместо этого можно использовать `import.meta.glob(GLOB_STR)` для достижения аналогичного результата.

## Что пошло не так?

`Astro.glob()` можно использовать только в файлах `.astro`. Для достижения того же результата можно использовать [`import.meta.glob()`](https://vite.dev/guide/features.html#glob-import).
