:::caution[Устарело]
Утратило актуальность с версии Astro 4.0. Вспомогательная функция RSS больше не существует с обратным вызовом ошибки.
:::

> **GetStaticPathsRemovedRSSHelper**: Вспомогательная функция RSS была удалена из `getStaticPaths`. Попробуйте вместо неё использовать новый пакет @astrojs/rss.

## Что пошло не так?

`getStaticPaths` больше не предоставляет вспомогательного инструмента для генерации RSS-канала. Мы рекомендуем перейти к использованию интеграции [@astrojs/rss](/ru/recipes/rss/#setting-up-astrojsrss).
