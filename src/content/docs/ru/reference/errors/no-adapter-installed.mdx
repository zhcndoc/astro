> **NoAdapterInstalled**: Невозможно использовать `output: 'server'` или `output: 'hybrid'` без адаптера. Пожалуйста, установите и настройте соответствующий серверный адаптер для вашего окончательного развертывания.

## Что пошло не так?

Для использования серверного рендеринга необходимо установить адаптер, чтобы Astro знал, как генерировать правильный вывод для вашей целевой платформы развертывания.
