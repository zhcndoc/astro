> **ImageMissingAlt**: Отсутствует свойство "alt" для изображения. "Alt" текст необходим для описания важных изображений на странице.

## Что пошло не так?

Свойство `alt` позволяет предоставить описательный текст для пользователей, пользующихся скринридерами и другими вспомогательными технологиями. Для обеспечения доступности изображений компонент `Image` требует указания атрибута `alt`.

Если изображение является исключительно декоративным (то есть не способствует пониманию страницы), задайте `alt=""`, чтобы скринридеры знали, что нужно игнорировать изображение.
