> Маршрут вернул `RETURNED_VALUE`. Из файлов Astro может быть возвращен только Response.

## Что пошло не так?

В файлах Astro может быть возвращен только объект [Response](https://developer.mozilla.org/en-US/docs/Web/API/Response).
```astro title="pages/login.astro"
---
return new Response(null, {
 status: 404,
 statusText: 'Not found'
});

// В качестве альтернативы, для перенаправлений Astro.redirect также возвращает экземпляр объекта Response.
return Astro.redirect('/login');
---
```
